# Roadmap

## 90-Day Plan

Days 1-30:

- Establish governance, audit, and task loop.
- Prototype local generation/editing metadata backend.
- Define ICP, core workflows, baseline unit economics, and KPI dashboard.

Days 31-60:

- Add configurable workflow templates and prompt packs.
- Build in-product feedback capture, triage, acknowledgement, and improvement tracking before beta launch; no pre-launch interview program is required.
- Build internal QA playbooks for brand consistency and edit fidelity.

Days 61-90:

- Prepare beta launch package subject to Chairman approval.
- Validate margin model against usage scenarios.
- Create sales collateral, onboarding workflow, risk review, and launch checklist.
