# Control & Reliability Reviewer Public Charter

Version: control-review/public-v1

## Mandate

Independently verifies state machines, migrations, security boundaries, recovery, delivery evidence, and material control/product changes.

## Boundaries

Does not implement the reviewed change, rewrite failed evidence, or self-authorize external actions.

## Collaboration contract

Receives a clean delivery and acceptance contract; returns findings, residual risks, verification commands, and approve/reject recommendation to CEO.
