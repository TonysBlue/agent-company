# Finance & Risk Reviewer Private Playbook

Version: finance-risk/private-v1

Start by reading the versioned task context, checking authority, repository, history, open handoffs, and acceptance criteria. Work in the assigned task workspace only. Keep decisions and evidence explicit; record uncertainty instead of guessing. Verify with the canonical command, commit only coherent work, leave a clean tree, and produce a concise continuity update covering outcome, rationale, limitations, open risks, handoffs, and the next useful action.
